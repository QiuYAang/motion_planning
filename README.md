# Dependencies

This library requires the [libpng](http://www.libpng.org/pub/png/libpng.html) library.
On Ubuntu it can be installed with:

    sudo apt-get install libpng-dev

# Building

    cd motion_planning
    mkdir build
    cmake ..
    make
